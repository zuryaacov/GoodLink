# מדריך משתמש מלא ל-GoodLink

## התחלה מהירה

1. היכנסו ל-`/login` והתחברו או צרו חשבון.
2. עברו ל-`/dashboard/links`.
3. לחצו `New Link` ובנו קישור חדש.
4. העתיקו את הקישור הקצר ועקבו אחרי התוצאות ב-`/dashboard/analytics`.

## דפים ציבוריים

- `/` - דף הבית
- `/login` - התחברות/הרשמה/איפוס סיסמה
- `/contact` - יצירת קשר
- `/abuse` - דיווח Abuse/DMCA
- `/terms` - תנאי שימוש
- `/privacy` - מדיניות פרטיות
- `/subprocessors` - ספקי משנה
- `/dpa` - DPA

## דשבורד

- `/dashboard` - אנליטיקה
- `/dashboard/links` - ניהול קישורים
- `/dashboard/utm-presets` - ניהול UTM
- `/dashboard/pixels` - ניהול CAPI
- `/dashboard/domains` - ניהול דומיינים
- `/dashboard/settings` - הגדרות חשבון
- `/dashboard/admin` - פאנל מנהל (אדמין בלבד)

## Link Wizard (יצירת קישור)

שלבים עיקריים:

1. Name - שם פנימי לקישור.
2. Target URL - כתובת יעד.
3. Domain (Advanced/Pro) - בחירת דומיין.
4. Slug - סיומת קישור.
5. Bot Protection (Advanced/Pro) - Allow/Block/Redirect.
6. Geo Targeting (Pro) - חוקים לפי מדינה.
7. CAPI Select (Pro) - חיבור פרופילי CAPI.
8. Review - סיכום ושמירה.

## UTM Preset Wizard

שלבים:

1. Name
2. Platform
3. Source
4. Medium
5. Campaign
6. Content
7. Term

אפשרויות UTM נבנות בהתאם לפלטפורמה, עם תצוגה מקדימה חיה.

## CAPI Wizard

שלבים:

1. Name
2. Platform
3. Pixel ID
4. CAPI Token
5. Event Type

יש ולידציה לפי פלטפורמה (Meta/TikTok/Google וכו').

## Domain Wizard

שלבים:

1. Domain
2. Root Redirect (אופציונלי)
3. DNS Setup
4. Verify

אם האימות לא עובר מיד, יש להמתין propagation ולנסות שוב.

## טיפים מומלצים

- שמרו Naming Convention עקבי לקישורים.
- השתמשו ב-UTM Presets במקום כתיבה ידנית.
- חברו CAPI לקמפיינים חשובים.
- בצעו בדיקת Analytics שוטפת.
