# הרצת בדיקות QA בדפדפן

הבדיקות האוטומטיות שמריצות את לוגיקת הוולידציה **בדפדפן** (עם Playwright) מוגדרות בתיקייה `tests/`.

## דרישות

- Node.js (גרסה נתמכת על ידי הפרויקט)
- התקנת Playwright (פעם אחת):

```bash
npm install
npx playwright install
```

## הרצה

- **הרצת כל בדיקות הדפדפן** (השרת יופעל אוטומטית אם צריך):

```bash
npm run test:qa:browser
```

או:

```bash
npx playwright test
```

- **הרצה עם ממשק גרפי** (פותח חלון שבו אפשר לראות את הבדיקות רצות):

```bash
npx playwright test --ui
```

- **הרצה בדפדפן גלוי** (לא headless):

```bash
npx playwright test --headed
```

## מה נבדק כרגע

- **דף Login** – שליחת טופס ריק, אימייל לא תקין, סיסמה שגויה (הודעות שגיאה).
- **דף Sign up** – וידוא שהשדות (שם, אימייל, סיסמה, אימות סיסמה) מופיעים, וכפתור "Create Account" מושבת עד השלמת Turnstile.
- **שכחתי סיסמה** – וידוא שדה אימייל וכפתור "Send Link", ושליחת אימייל לא תקין מחזירה הודעת וולידציה.
- **ניווט** – מעבר בין Login ל-Sign up וחזרה.

## בדיקות וולידציה מלאות (ללא דפדפן)

לוגיקת וולידציה (אימייל, סיסמה, slug, pixel, domain, XSS וכו') נבדקת גם ב-Node:

```bash
npm run test:qa
```

זה מריץ את `qa-test-runner.mjs` ומכסה את כל הפונקציות מ-`QA-INPUT-VALIDATION.md`.
